# 📊 Super Store Sales Dashboard

![Dashboard Preview](images/dashboard.png)

## Overview

An interactive Power BI dashboard designed to analyze sales performance, profit trends, customer behavior, and regional performance. The dashboard transforms raw sales data into actionable business insights through visual analytics and KPI tracking.

---

## Key KPIs

- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Orders
- 👥 Customer Segments
- 🌎 Regional Performance
- 🏆 Top Products

---

## Dashboard Features

### Sales Analysis
- Monthly Sales Trend
- Year-over-Year Growth
- Category-wise Sales

### Profit Analysis
- Profit by Category
- Profit Margin Tracking
- Loss-Making Products

### Regional Insights
- Region-wise Sales
- State Performance
- Geographic Analysis

### Product Insights
- Top Selling Products
- Best Performing Categories
- Product Contribution Analysis
